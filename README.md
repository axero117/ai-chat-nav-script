# 📑 AI对话导航增强版

一个为多个 AI 对话网站提供**消息快速导航、对话目录搜索、对话导出和内容宽度调整**的油猴脚本。侧边栏悬浮展开，支持键盘快捷键，让聊天体验更高效。

---

## ✨ 功能亮点

### 快速跳转消息
点击右侧边栏的 **↑ / ↓** 按钮，或使用快捷键 `Ctrl + Alt + ↑ / ↓`，快速定位到当前对话的上一条/下一条用户消息。

### 目录面板（消息 + 对话）
点击 **📑** 按钮或按 `Ctrl + Alt + D` 打开目录面板，可在“消息”和“对话”标签页间切换，并支持关键词搜索快速定位。

- **消息目录**：列出当前对话中所有用户消息，点击后滚动到对应位置并高亮显示。
- **对话目录**：列出左侧所有历史对话，点击后直接跳转到该对话。

### 导出对话
点击 **📤** 按钮或按 `Ctrl + Alt + E`，可将当前对话的所有用户消息导出为 Markdown 文件，方便存档或分享。

### 内容宽度自由调整
点击 **➕/➖** 按钮或按 `Ctrl + Alt + ← / →`，以 50px 为步长增加/减少聊天区域宽度（可从 600px 调整到 2000px）。点击 **↻** 或按 `Ctrl + Alt + R` 恢复默认宽度（900px）。

### 悬浮展开侧边栏
侧边栏默认只显示图标，**鼠标悬浮时展开显示完整文字**，节省屏幕空间，需要时一目了然。侧边栏底部新增 **❓** 按钮，点击可查看所有快捷键说明。

### 多平台支持
目前已适配六个主流 AI 对话网站：
- [ChatGPT](https://chatgpt.com/)
- [Gemini](https://gemini.google.com/)
- [豆包](https://www.doubao.com/chat/)
- [DeepSeek](https://chat.deepseek.com/)
- [千问](https://www.qianwen.com/chat)
- [Kimi](https://www.kimi.com/chat/)

## ⌨️ 键盘快捷键

| 快捷键                | 功能                 |
|----------------------|----------------------|
| `Ctrl + Alt + ↑`     | 上一条用户消息       |
| `Ctrl + Alt + ↓`     | 下一条用户消息       |
| `Ctrl + Alt + D`     | 打开/关闭目录面板    |
| `Ctrl + Alt + E`     | 导出对话             |
| `Ctrl + Alt + ←`     | 减小内容宽度         |
| `Ctrl + Alt + →`     | 增加内容宽度         |
| `Ctrl + Alt + R`     | 重置内容宽度为 900px |

> ⚠️ **注意**：快捷键仅在未聚焦输入框/文本域时生效。

## 📦 安装方法

1. **安装油猴扩展**  
   在浏览器中安装 [Tampermonkey](https://www.tampermonkey.net/) 或 [Violentmonkey](https://violentmonkey.github.io/) 扩展。

2. **获取脚本代码**  
   访问 [GitHub 仓库](你的GitHub仓库链接)，打开 `dialog-navigator.user.js` 文件，**复制文件中的所有代码**。

3. **创建新脚本**  
   - 点击浏览器右上角的油猴图标，选择 **“添加新脚本”**（或 **“创建新脚本”**）。
   - 删除编辑器中默认的代码，将复制的脚本内容**粘贴进去**。
   - 按 `Ctrl + S` 保存（或点击菜单 **文件 → 保存**）。

4. **刷新网页**  
   打开任意支持的对话网站，刷新页面即可生效。

## 📝 使用提示

- 首次安装后，请**刷新**对应网站页面以加载脚本。
- 如果对话目录未正确显示，请尝试刷新页面或检查控制台日志（`F12` → Console）查看错误信息，并反馈给作者。
- 宽度调整功能会自动保存你的设置，下次打开网站时自动恢复。
